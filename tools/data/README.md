# tools/data (Release)

This release includes lightweight dataset metadata/config files only.

Included:
- `prepare_datasets.py`
- `loaders.py`, `transforms.py`, `text_utils.py`
- `configs/data_paths.yaml`
- `flickr30k/karpathy_splits.json`
- `flickr30k/results_20130124.token`

Not included in GitHub (too large / license-gated): raw image archives and extracted datasets.
Use `prepare_datasets.py` and the project README to download/prepare data locally.
